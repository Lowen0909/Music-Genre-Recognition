## Dataset 
[GTZAN](https://www.kaggle.com/datasets/andradaolteanu/gtzan-dataset-music-genre-classification)
10 genres with 100 audio files each, all having a length of 30 seconds
Sampling rate: 22050Hz
Categories: Blue, Classical, Country, Disco, Hiphop, Jazz, Metal, Pop, Reggae, Rock
